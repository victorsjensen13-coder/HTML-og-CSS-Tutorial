Hvordan farver jeg min hjemmeside?

Over din <title> skriv "<lnk rel="stylesheet" href="[you .css file name]> åbn din css fil så du har din html fil og css fil åben. I din css, skrive
"button {
         background-color: red;
}"
Nu at du har skrevet det, kan du gemme dine filere og åben din index.html. Nu er backgrounden rød. :D
Hvis du ikke forstår hvor vi skriver "button" er det fordi den type vi ændre er en button. Hvis du og så ikke forstår det har "{}" er det basicly der vi setter de ting vi skal farve eller andet.
Hvis du gerne ville lærer mere af de forskellige probaties (som fx background-color) så klik på denne knap for en liste.
