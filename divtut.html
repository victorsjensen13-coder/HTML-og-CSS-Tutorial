Divs

is a way to make boxes with info it in. We can also use CSS to color it, animate it, transform it(scale, rotate etc) but we can start of in the CSS where we type this
"div {
background-color: lime;
width: 200px;
height: 100px;
border: 2px solid black;
}"
What this does it that it sets the color to lime, sets the width to 200px, the height to 100px and the border to 2px solid black. Here's how it looks

Next is Transforming it.


In our css we will make it look like this
"div {
background-color: lime;
width: 200px;
height: 100px;
border: 2px solid black;
transition-duration: 0.5s;
transform: all;
}"
This will rotate the div 20 degrees and scale it to 1.5 times its size. Now go some lines down (outside the div {}) now make a "div:hover". We should make it look like this.
div:hover {
transform: rotate(20deg) scale(1.5);
}
Which whould make it look like this when you hover over it (try hovering over the box) it is important that "transform: rotate(20deg) scale(1.5);" (the rotate and scale) are next to each other. It doesn't matter in which order.
